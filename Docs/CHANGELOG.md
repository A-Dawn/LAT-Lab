## [v1.0.1] - 2025-08-15

### 安全更新
- 🔒 修复了API中允许普通用户获得管理员权限的权限提升漏洞
- 🛡️ 增强了用户角色更改的安全验证
- 🔐 为安全相关操作添加了额外的日志记录

### 错误修复
- 📝 修复了文章编辑器在使用特殊字符时的保存问题
- 🖼️ 修复了管理面板中图片上传的错误处理
- 🔍 修复了移动端视图中的搜索功能

### 改进优化
- ⚡ 提升了文章列表页面的加载性能
- 🧩 增强了插件系统的稳定性
- 📱 优化了小屏幕设备的响应式布局

### 文档更新
- 📚 添加了 SECURITY_CONTRIBUTORS_EN.md 文件
- 📝 更新了贡献者文档
